# DBF 2026 — Manufacturing Schedule

A concurrent-track manufacturing schedule for Cornell's AIAA Design Build Fly 2026 team. Hosted via GitHub Pages.

## Live Site
Once deployed: `https://<your-username>.github.io/<repo-name>/`

## Deploy to GitHub Pages

1. **Create a new GitHub repo** (e.g. `dbf-schedule`)

2. **Push these two files to the repo root:**
   ```
   index.html
   README.md
   ```

3. **Enable GitHub Pages:**
   - Go to repo → **Settings** → **Pages**
   - Under *Source*, select **Deploy from a branch**
   - Branch: `main` | Folder: `/ (root)`
   - Click **Save**

4. Your site will be live at `https://<username>.github.io/dbf-schedule/` within ~60 seconds.

## Updating the schedule
Replace `index.html` with the new version and push. GitHub Pages redeploys automatically.
